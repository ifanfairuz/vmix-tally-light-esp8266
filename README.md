# vMix Tally Light ESP8266
Arduino ESP8266 vMix Tally Light

### Features
- Custom Config
- Web Based Config

### Quick Start
1. Flash ESP8266 with arduino.ino
2. Wait & Check your WiFi
3. Connect to TallySource{number} SSID
4. Go to `http://192.168.2.1`
5. Setting your TallyLight
6. Save Setting

### Tested on
- LOLIN(WEMOS) D1 Mini
